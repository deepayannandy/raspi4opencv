# raspi4opencv
Easiest and effective way to build and install open cv in a raspberry pi 4 computer


```bash
$ git clone 
$ sudo su
$ chmod +x *.sh
$ ./download-opencv.sh
$ apt-get install vlc-plugin-qt
$ ./install-deps.sh
$ ./build-opencv.sh
$ cd ~/opencv/opencv-4.1.2/build
$ sudo make install
```
